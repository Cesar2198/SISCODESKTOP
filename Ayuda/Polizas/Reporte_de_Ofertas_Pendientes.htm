REPORTE DE OFERTAS PENDIENTES

 

Genera un reporte con las ofertas pendientes de ser aceptadas, las ofertas que se toman en cuenta son las ofertas individuales ingresadas al sistema y las ofertas comparativas.

 

 


DESCRIPCIÓN DE LOS CAMPOS:

Fecha de corte:                Selecciona la fecha de corte que se asignará para la búsqueda de ofertas pendientes de enviar.

Buscar Ofertas:                Utilice éste botón para ejecutar la orden de búsqueda de las ofertas.

 

MODO DE OPERAR

Seleccione las fechas de corte para asignarla a la búsqueda de las ofertas pendientes, por último utilice el botón “Imprimir” para generar el reporte con la información de las ofertas pendientes.
