CONDICIONES ESPECIALES

 

Permite Crear, modificar o eliminar las condiciones especiales de pólizas, cabe mencionar que el sistema SISCO trae previamente cargada una lista de éstas.

 


DESCRIPCIÓN DE LOS CAMPOS:

Cód. Condición: Ingrese el Código con el cual el sistema reconocerá la condición especial.

Ramo:   Elija el ramo al cual se podrá emplear dicha condición especial.

Descripción: Agregue una breve descripción en la cual consiste la condición especial de póliza.

 

MODO DE OPERAR

Llene o modifique los campos antes mencionados y utilice el botón “Guardar” para que la modificación o la nueva condición especial quede guardada.
