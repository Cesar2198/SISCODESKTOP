COMPARATIVO OFERTAS

 

Permite crear un reporte en Excel con la finalidad de analizar la oferta y compararla con otra oferta según la aseguradora que la ofrece, el mecanismo funciona por puntajes que se les especifican a cada cobertura, cláusula o condición sobre cada oferta  de las aseguradoras a fin de ponderar y proponer la mejor de las ofertas al cliente.

 

 

 

DESCRIPCIÓN DE LOS CAMPOS:

ID Comparativo:  Ingresa un código con el cual se identificará la oferta a comparar.

Estado Oferta: Selecciona el estado en el que se encuentra la oferta sea ésta pendiente, aceptada o rechazada.

Cod. Cliente: Muestra el código del cliente, éste es el número de N.I.T.

Buscar Cliente: Botón de búsqueda de clientes para cargar la información previamente guardada del cliente.

Nombre Cliente: Muestra el nombre completo del cliente.

Ramo de Seguro: Elija el ramo al cual se podrá emplear dicha condición especial.

Ejecutivo Cta.:  Asigna al ejecutivo de cuenta que está asociado al cliente, dichos ejecutivos deberán de estar ingresados previamente para poder seleccionarlo.

Fecha Oferta: Selecciona la fecha en la que ha sido creada la oferta.

Aseguradora:Selecciona la aseguradora en la cual está basada la oferta.

 

MODO DE OPERAR

Ingrese los datos requeridos por los campos antes descritos, al seleccionar el ramo de seguro le cargará una lista con las diferentes coberturas, cláusulas especiales, primas y gastos, deducibles y número de pagos, usted podrá utilizar el botón “Agregar Referente” para incorporar más o utilizar el botón “Eliminar Referente” para eliminar algunos de los que se tengan.

Importante mencionar que el botón MAS nos servirá para agregar las aseguradoras que se compararán sus ofertas y utilizar el botón MENOS para quitar una aseguradora de la comparación de ofertas.

Una vez puestas las aseguradoras con las que se compararán las ofertas usted podrá ir marcando las coberturas que una aseguradora proporciona y así poder compararla con la otra.

Finalmente utilice el botón “Imprimir” para generar el reporte comparativo en el programa de Excel.
