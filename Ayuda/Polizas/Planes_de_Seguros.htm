PLANES DE SEGUROS

 

Permite elaborar planes de seguros que serán utilizados opcionalmente al ingresar las coberturas de los certificados, esto permite tener un paquete previamente personalizado con diferentes Suma Asegurada, % Suma, Tasa, Prima, Deducible, % Suma R1 en el momento de ingresar las coberturas para un cliente en especial.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

Plan: Ingrese el nombre que le asignará al plan.

Descripción: Ingrese una breve descripción de lo que consiste el plan a crear.

Ramo: Seleccione de la lista previamente cargada el ramo al cual pertenecerá el plan a crear.

Producto: Seleccione de la lista previamente cargada el producto al cual pertenecerá el plan a crear.

 

MODO DE OPERAR

Seleccione un nombre para el plan a crear, agregue una breve descripción en el cual consistirá el plan, elija el ramo y producto al que pertenecerá el plan, luego le cargará las coberturas en el cual usted deberá de proporcionar las cantidades o porcentajes para la Suma Asegurada, % Suma, Tasa, Prima, Deducible, % Suma R1, por último utilice le botón “Guardar” para que sea ingresado al sistema SISCO el nuevo plan de seguro.
