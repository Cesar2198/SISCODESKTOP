PLANES SINIESTROS

 

Los planes de siniestros es una manera de agrupar a un determinado número de reclamos ya sea por un producto o una póliza en particular, la idea es que a estos reclamos se les puede dar una numeración especial para diferenciarlos.

 

 

 

DESCRIPCIÓN DE LOS CAMPOS:

 

Buscar Póliza:                   Realiza una búsqueda de póliza.

 

Cód. Plan: Ingrese el código al plan se está creando.

Descripción: Campo para ingresar una breve descripción del plan de reclamos.

Póliza: Muestra el número de póliza.

Producto: Muestra el producto del cual hace referencia la póliza.

Patón Número Reclamo: Ingrese el patrón número reclamo, siguiendo las indicaciones para tal efecto:

Patrones a Validar = #: Correlativo por plan, yy#: Correlativo Año Plan.

 

 

MODO DE OPERAR

 

Utilice el botón “Buscar Póliza” para seleccionar la póliza que se le quiere aplicar el plan de reclamos, esto cargará la información necesaria a lo que deberá de revisar que dicha información sea la correcta y completar los campos que se requieran.

 

Hecho lo anterior deberá de utilizar el botón “Guardar” para que el plan de reclamos quede guardado.
