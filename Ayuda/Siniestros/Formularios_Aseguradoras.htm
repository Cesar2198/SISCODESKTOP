FORMULARIOS ASEGURADORAS

 

Guarde en el sistema las rutas donde se tienen almacenados los formularios que en todas las aseguradoras utilizan ya sean estos de Póliza, Reclamos o de Cartera; una vez guardadas en el sistema podrá posteriormente revisarlos rápidamente.

 

 

 

DESCRIPCIÓN DE LOS CAMPOS:

Cód. Formulario: Ingrese un código único con el que se identificarán los formularios.

Descripción: Ingrese una breve descripción del formulario.

Aseguradora:  Seleccione la aseguradora a que pertenece el formulario.

Tipo Formulario: Seleccione el tipo de formulario al que pertenece.

Ruta Formulario: Muestra la ruta en la que se encuentra guardado el formulario.

 

Buscar Archivo:  Utilice éste botón para buscar el archivo que contiene el formulario.

 

MODO DE OPERAR

Completar los campos correspondientes y utilice el botón “Buscar Archivo” para que incorpore la ruta en la que se tiene guardado el archivo que contiene el formulario y por último utilizar el botón “Guardar” para completar la operación.

 

Posteriormente podrá buscar el archivo que contiene el formulario para poder ver el formulario.
