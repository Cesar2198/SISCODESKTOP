Generar Documentos

La generación de documentos permite la impresión de todo tipo de cartas que se dirigen a los clientes o a las aseguradoras, además de generar el documento este se liga a un flujo existente o se puede iniciar uno nuevo desde esta mismo programa.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

 

Documento: Acá se selecciona de la lista el documento que se desea generar

Cod. Cliente: Es el código del cliente en el sistema

Nombre Cliente: Nombre del cliente sobre el que se le generará el documento

Flujo: Flujo del cliente al que se le asignará el documento, sino existe en la lista de flujo existe la opción de crear uno nuevo

Fecha en Documento: Fecha que se especifica en el documento a generar

 

 

MODO DE OPERAR

 

Cuando se especifique el documento y el cliente automáticamente se propondrán los documentos del sistema relacionados al cliente y al documento, si es una carta de renovación por ejemplo se mostrarán las pólizas del cliente, seleccione un registro relacionado para que sea ingresado en el documento y presionar el botón imprimir.
