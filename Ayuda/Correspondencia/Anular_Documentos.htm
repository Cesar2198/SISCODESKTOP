Anular Documentos

Cuando en el sistema de correspondencia se genera un documento se genera un número único de control y seguimiento del documento y si el documento generado contienen errores o no procede la entrega del documento se puede anular este código y además la correspondencia no se entregará.

 

 

 

 

MODO DE OPERAR

 

Debe especificar la fecha en la que se ha generado el documento, por defecto el programa propone la fecha actual, se listarán los documentos generados ese día y únicamente debe seleccionar el documento que desee anular luego aparecerá una pantalla que le solicitará la fecha de anulación y el motivo por el cual se anula el documento.
