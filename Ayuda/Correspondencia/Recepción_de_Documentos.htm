Recepción de Documentos

La recepción de documentos permite registrar un documento recibido ya sea de un cliente, aseguradora o proveedor y que se registra bajo un flujo ya iniciado o se puede iniciar uno.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

 

Cod. Cliente: Código del cliente sobre el que se recibe la correspondencia

Nombre Cliente: Nombre del cliente de la correspondencia que se recibe

Descripción del Documento: Una descripción detallada del documento que se recibe

Flujo: Se debe especificar el flujo o crear uno desde este mismo campo

Fecha en Documento: Se especifica la fecha que presenta el documento que se recibe

Destinatario: Departamento al que va dirigido el documento que se recibe.

 

 

MODO DE OPERAR

 

Se deben especificar todos los datos obligatorios sobre la correspondencia que se recibe, es importante especificar el flujo al que se asigna la correspondencia para tener un control de todos los documentos de un procedimiento.
