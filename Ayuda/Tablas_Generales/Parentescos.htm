PARENTESCOS

Podrá crear, modificar o eliminar el parentesco, previamente el sistema trae cargado los parentescos existentes.

 

 

 

DESCRIPCIÓN DE LOS CAMPOS:

Cód. Parentesco: Es el código único con el que se clasifica el parentesco dentro del sistema.

Descripción: Podremos dar una breve descripción del Código que ingresamos.

 

MODO DE OPERAR

Ingresar correctamente los datos que se requiere para cada campo, hecho esto deberá de utilizar el botón “Guardar”, para que el registro de parentesco sea ingresado al sistema.
