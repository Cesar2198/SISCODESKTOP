PRODUCTOS DE SEGUROS

Podrá crear, modificar o eliminar los diferentes productos de seguros, el sistema ya los trae cargados previamente para que se puedan duplicar (Botón “Guardar Como”) y posteriormente personalizar para ser utilizados en diferentes Aseguradoras.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

 

GENERAL

Cód. Producto: Es el código único con el que se clasifica dentro del sistema, un determinado Producto de Seguros.

Descripción: Podremos dar una breve descripción del Código que ingresamos.

Aseguradora: Seleccione del combo la Aseguradora a la cual pertenece el Producto.

Tipo de Producto: Seleccione del combo el tipo de producto entre Vida o Daños.

Ramo de Seguro: Seleccione del combo el ramo de seguro al que pertenece el producto.  

Cód. Producto en Aseguradora: Ingrese el código del producto con el cual se identifica en la aseguradora.

 

COBERTURAS

Cobertura: Cantidad de coberturas asignadas al producto de seguros.

Nombre Cobertura: Nombre de cada una de las coberturas asignadas al producto de seguros.

Tarificación: Porcentaje de tarifa por cobertura.

Descuento: Indica si existe o no algún tipo de descuento.

Cód. Cob. En Aseg.: Indica el código de cobertura que asigna la aseguradora.

 

GASTOS

Cód. Gasto: Selecciona del combo el tipo de gasto al que se refiere.

Tipo Cálculo: Selecciona del combo el tipo de cálculo en que se realizara el cobro del gasto, ya sea por prima o por suma.

%. Gasto: Se asigna el porcentaje del gasto según el tipo de gasto al que se refiere.

Valor Gasto: Se asigna el valor del gasto según el tipo de gasto al que se refiere.

Valor Mínimo: Se asigna el valor Mínimo del gasto según el tipo de gasto al que se refiere.

Valor Máximo: Se asigna el valor Máximo del gasto según el tipo de gasto al que se refiere

Cód. Aseguradora: Código que le asigna la aseguradora al gasto.

 

COMISIONES

Comisiones recibidas por las aseguradoras.

Año: Hace referencia al primer año o a posteriores en el cual se recibirán las comisiones por parte de las aseguradoras.

% Comisión: Porcentaje de comisión que recibirán por parte de las aseguradoras.

Valor Fijo: Ingresa la cantidad cuando se trate de una comisión por un valor fijo.

% Gas. Admin.: Porcentaje de gasto administrativos que genere la comisión.

Forma de Pago: Forma de pago en la que la correduría recibirá por parte de las aseguradoras las comisiones ya sean éstas periódica en el pago con cada factura o anticipada de un solo pago.

 

COMISIONES OTORGADAS A LOS VENDEDORES

Id Comisión: Identificación de la comisión.

Año: Hace referencia al primer año o a posteriores en el cual se entregarán al vendedor las comisiones por parte de la correduría.

Tipo Vendedor: Selecciona del combo el tipo de vendedor ya sea este interno, externo o asociado.

% Comisión: Asignación del porcentaje de comisión para vendedores.

Valor Comisión: Ingresa la cantidad cuando se trate de una comisión por un valor fijo.

Forma de Pago: Forma de pago en la que la correduría entregará a los vendedores las comisiones ya sean éstas periódica en el pago con cada factura o anticipada de un solo pago.

 

DATOS TÉCNICOS PRODUCTO

Código: Código asignado a la descripción de bienes.

Descripción de Bienes: Descripción de bienes asegurados.

 

SINIESTROS

Requisitos para presentar reclamos.

Orden: Número de orden ingresada.

Descripción de Requisitos: Ingresar una breve descripción del requisito al que se refiere dicha orden.

 

Datos complementarios declaración reclamos.

Orden: Número de orden ingresada como complemento a la declaración del reclamo.

Descripción Dato Complementario: Ingresar una breve descripción del dato complementario al que se refiere dicha orden.

Categoría: Ingrese la categoría a la cual pertenece la orden.

 

Tipos de gastos en reclamos.

Orden: Número de orden ingresada como gasto en reclamo.

Descripción: Ingresar una breve descripción del gasto al que se refiere dicha orden.

Cob. Afecta: Cobertura a la que afecta dicho gasto.

Cod. Aseg.: Código asignado por la Aseguradora correspondiente.

 

MODO DE OPERAR

Ingresar correctamente los datos que se requiere para cada campo con la información del producto de seguro, tomar en cuenta los campos asignados en las secciones: General, Coberturas, Gastos, Comisiones, Datos técnicos producto y siniestros. Hecho esto deberá de utilizar el botón “Guardar”, para que el registro del Producto del Seguro sea ingresado al sistema.

NOTA: Utilizar el botón “Guardar como” en los casos que se quiera duplicar un producto para ser utilizado en otra Aseguradora distinta.
