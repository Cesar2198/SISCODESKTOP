BANCOS

Permite el ingreso de registro de los diferentes bancos existentes dentro del sistema financiero.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

Cód. Banco: Es el código único con el que se clasifica dentro del sistema, un determinado Banco.

Nombre Banco: Ingresa el nombre del Banco.

Teléfono: Número de teléfono principal del Banco.

 

MODO DE OPERAR

Ingresar correctamente los datos que se requiere para cada campo con la información del Banco, hecho esto deberá de utilizar el botón “Guardar”, para que el registro de Banco sea ingresado al sistema.
