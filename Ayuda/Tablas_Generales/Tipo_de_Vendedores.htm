TIPO DE VENDEDORES

Podrá crear, modificar o eliminar el tipo de vendedor con los que la correduría trabajará, previamente el sistema trae cargado los tipos de vendedores internos, externos y corredor asociado.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

Cód. Tipo Vendedor: Es el código único con el que se clasifica el Tipo de vendedor dentro del sistema.

Descripción: Podremos dar una breve descripción del Código que ingresamos.

 

MODO DE OPERAR

Ingresar correctamente los datos que se requiere para cada campo, hecho esto deberá de utilizar el botón “Guardar”, para que el registro de Tipo de Vendedor sea ingresado al sistema.
