TIPOS DE GASTOS

Podrá crear, modificar o eliminar los tipos de gastos que generan los seguros, previamente el sistema trae cargado los tipos de gastos de bombero, emisión, fraccionamiento u otros gastos.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

Cód. Gasto: Es el código único con el que se clasifica dentro del sistema, el gasto de seguro.

Nombre de Gasto: Nombre al que se le asigna al tipo de gasto que genera el seguro.

Forma Pago: Indica la forma en la que se pagará el gasto, si es fraccionado o en la primera cuota.

 

MODO DE OPERAR

Ingresar correctamente los datos que se requiere para cada campo, hecho esto deberá de utilizar el botón “Guardar”, para que el registro de Tipo de gastos sea ingresado al sistema.
