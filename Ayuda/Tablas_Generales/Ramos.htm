RAMOS DE SEGUROS

De acuerdo al riesgo que cubren según los denominados Ramos de Seguros, podrá crear, modificar o eliminar los diferentes ramos que el sistema trae cargados previamente.

 

 

DESCRIPCIÓN DE LOS CAMPOS:

Cód. Ramo: Es el código único con el que se clasifica dentro del sistema, un determinado Ramo de Seguro.

Nombre Ramo: Ingresa el nombre del Ramo de Seguro.

Grupo Ramo: Selecciona del combo el grupo al cual pertenece el ramo de seguro, ya sea que se trate de Autos, Daños, Fianzas o Personas.

 

MODO DE OPERAR

Ingresar correctamente los datos que se requiere para cada campo con la información del Ramo del Seguro, hecho esto deberá de utilizar el botón “Guardar”, para que el registro del Ramo del Seguro sea ingresado al sistema.
